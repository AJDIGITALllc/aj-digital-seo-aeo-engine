## Purpose

Anti-Gravity generates visual UI from text prompts. These prompts are tuned to produce output that aligns with AJ Digital's design system, conversion strategy, and quality standards. They encode design decisions that Anti-Gravity needs to produce premium results.

## Website Generation Prompts

### Full Marketing Website

```
Generate a premium marketing website for [CLIENT NAME], a [INDUSTRY] company that [WHAT THEY DO].

TARGET AUDIENCE: [WHO THEY SERVE — be specific about demographics, role, pain points]

BRAND DIRECTION:
- Primary color: [HEX or descriptive, e.g., "deep navy blue"]
- Accent color: [HEX or descriptive]
- Style: [modern minimalist / bold and energetic / elegant and refined / corporate professional]
- Typography feel: [clean sans-serif / editorial serif / geometric modern]
- Imagery: [photography style — dark and moody / bright and airy / corporate / lifestyle]

PAGES TO GENERATE:
1. Homepage — hero, services overview, testimonials, results, CTA
2. Services page — detailed service offerings with benefits
3. About page — team, mission, company story
4. Contact page — form, map, contact details

DESIGN REQUIREMENTS:
- Generous white space — sections should breathe
- 8px spacing rhythm throughout
- Strong typographic hierarchy with clear heading progression
- Primary CTA visible above the fold on every page
- Navigation: clean, 5-7 items, CTA button in nav
- Footer: comprehensive with contact info, social links, sitemap
- Mobile-responsive with clean stacking behavior

CRO REQUIREMENTS:
- Every page has a clear primary action
- Trust signals (testimonials, logos, metrics) near conversion points
- FAQ section on service pages for objection handling
- Clear value proposition in hero — outcome-focused, not feature-focused
- Contact form: minimal fields (name, email, message, optional budget)

QUALITY:
- No stock photo look — direction should feel custom and intentional
- No cluttered layouts — every element earns its space
- Consistent visual language across all pages
- Premium feel appropriate for a $10K–$100K service
```

### Landing Page (Lead Generation)

```
Generate a high-converting lead generation landing page for [CLIENT NAME].

OFFER: [What the visitor gets — free consultation, audit, guide, demo]
AUDIENCE: [Who this is for — specific role, company size, industry]
GOAL: [Specific action — fill out form, book a call, download resource]

DESIGN DIRECTION:
- Style: [clean and modern / bold and conversion-focused / elegant and trustworthy]
- Primary color: [HEX]
- Background: [white / light gray / dark with light text]
- Mood: [confident / approachable / urgent / premium]

PAGE STRUCTURE:
1. HERO
   - Headline: benefit-driven, under 10 words
   - Subheadline: who it's for + the specific outcome
   - CTA button: large, contrasting, action verb
   - Trust line: "Trusted by X+ companies" or client logos
   - Visual: shows the outcome or the product/service

2. PROBLEM SECTION
   - 3 specific pain points the audience experiences
   - Written in their language, not marketing speak
   - Creates recognition: "that's exactly my problem"

3. SOLUTION SECTION
   - 3 pillars of how the offer solves the problem
   - Icon + headline + 2-sentence description each
   - Outcome-focused: what changes after using the service

4. SOCIAL PROOF
   - 2-3 testimonials with specific results
   - Name, title, company, photo
   - At least one testimonial includes a metric

5. HOW IT WORKS
   - 3-step process: simple, visual, numbered
   - Step 1: [action] → Step 2: [action] → Step 3: [outcome]
   - Reduces complexity and perceived effort

6. FAQ
   - 5-7 common questions
   - Each answer starts with a direct response
   - Handles objections: pricing, timeline, commitment, results

7. FINAL CTA
   - Restated value proposition
   - Large CTA button
   - Risk reversal: "No commitment" / "Free" / "Cancel anytime"

DESIGN RULES:
- Section spacing: generous, never cramped
- Visual hierarchy: headline → supporting text → CTA (clear eye path)
- One primary CTA per section — do not compete with multiple actions
- Alternating section backgrounds for visual rhythm
- Typography: large headlines (48-68px), comfortable body text (16-18px)
- Max content width: ~1280px with generous side padding
```

## Premium UI Direction

### Design Principles for Anti-Gravity Output

When generating UI, these principles determine whether the output feels "premium" or "generic":

**1. Intentional White Space**
- Sections need room to breathe — double the spacing you think you need, then evaluate
- Content blocks should never touch the edges of their containers
- The most important content gets the most space around it

**2. Typography as Design**
- Large, bold headlines create impact and hierarchy
- Size contrast between heading and body text should be dramatic (at least 2× difference)
- Limit to 2 font families maximum — one for headings, one for body

**3. Restrained Color**
- Use color sparingly — let it draw attention to CTAs and key elements
- 80% of the page should be neutral tones (whites, grays, blacks)
- When color appears, it should feel intentional, not decorative

**4. Visual Rhythm**
- Alternate section styles: white background → colored/gray background → white
- Consistent element sizing within sections (all cards same height, all icons same size)
- Predictable spacing creates a sense of craftsmanship

**5. Purposeful Imagery**
- Images should show outcomes, products, or real people — not abstract stock photos
- Dark overlays on hero images for text readability
- Consistent image treatment throughout (all photos same filter/style)

**6. Micro-Details**
- Subtle shadows instead of harsh borders
- Rounded corners (8px or 16px) for approachability
- Smooth hover states that feel responsive
- Consistent icon style (outline or filled — not mixed)

## CRO-Aware Design Instructions

Include these in any Anti-Gravity prompt to ensure conversion optimization:

```
CONVERSION OPTIMIZATION RULES:
- The primary CTA must be the most visually prominent interactive element on the page
- Trust signals (logos, testimonials, metrics) must appear before or alongside CTAs
- Form fields should be minimal — only ask for what's absolutely necessary
- The hero section must communicate: what it is, who it's for, and what to do next
- Visual hierarchy must guide the eye: headline → supporting text → CTA → trust
- No competing CTAs in the same viewport — one primary action per section
- Risk reversal (guarantee, "free", "no commitment") placed near the final CTA
- Social proof is visual — photos of real people, real company logos, specific numbers
```

---

## AEO-Ready Structure Instructions

Include these to ensure the generated site supports AI search optimization:

```
AI SEARCH OPTIMIZATION RULES:
- Every H2 section should answer an implicit question that a user might search for
- FAQ sections must use proper question-answer structure (not just styled text)
- The first sentence of every major section should be a clear, extractable statement
- Include "What is [topic]?" and "How does [process] work?" patterns in content
- Service descriptions should be specific and factual — avoid vague marketing language
- Include specific numbers, timeframes, and metrics where applicable
- Structure content so each section stands alone as a complete answer
```

## Design Style Guidance

### Modern Minimalist (Default AJ Digital Style)

```
STYLE: Modern minimalist
- Clean sans-serif typography (Inter, DM Sans, or system fonts)
- Generous white space with 8pt spacing system
- Neutral palette with one bold accent color
- Subtle shadows, no harsh borders
- Photography: bright, well-lit, intentional
- Layout: grid-based, lots of breathing room
- Animation: subtle, purposeful (fade in on scroll, smooth hover states)
```

### Bold and Energetic

```
STYLE: Bold and energetic
- Strong sans-serif with heavy headline weights
- High contrast: dark backgrounds with vibrant accent colors
- Dynamic layouts: asymmetric grids, overlapping elements (controlled)
- Photography: high-energy, action-oriented
- Bolder shadows and elevation
- Animation: slightly more dramatic (slide-in, scale-up)
```

### Elegant and Refined

```
STYLE: Elegant and refined
- Serif headings paired with sans-serif body
- Muted color palette with gold/copper/rich accent
- Extra-generous white space
- Thin borders, delicate lines
- Photography: editorial quality, muted tones
- Animation: slow, graceful transitions
```

### Corporate Professional

```
STYLE: Corporate professional
- Clean sans-serif throughout (no playful fonts)
- Blue/navy primary palette with structured neutrals
- Structured grid layouts, no asymmetry
- Professional photography: headshots, office, team
- Minimal animation: subtle fades only
- Data-forward: charts, metrics, process diagrams
```

## Output Validation Checklist

After generating any UI with Anti-Gravity, verify:

- [ ] Visual hierarchy is clear — you can identify the most important element in each section
- [ ] Spacing is generous and consistent — no cramped areas
- [ ] CTA buttons are visually prominent and positioned strategically
- [ ] Trust signals are present near conversion points
- [ ] Typography has clear contrast between headings and body
- [ ] Color usage is restrained — accent color draws eyes to key elements
- [ ] Layout is clean — no cluttered sections or competing visual elements
- [ ] Mobile responsiveness looks intentional (not just stacked desktop)
- [ ] The overall feel matches the intended brand direction
- [ ] Every section has a clear purpose in the conversion funnel
